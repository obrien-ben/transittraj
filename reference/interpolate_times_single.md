# Time interpolation for single trajectories

Not intended for external use.

## Usage

``` r
interpolate_times_single(trip_extremes, new_distances, inv_trajectory_function)
```

## Arguments

- trip_extremes:

  DF of max and min time values

- new_distances:

  DF of new distance points

- inv_trajectory_function:

  Inverse trajectory function

## Value

DF of interpolated values
